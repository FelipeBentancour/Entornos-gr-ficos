        <li>
            <strong>¿Qué es HTML, cuándo fue creado, cuáles fueron las distintas versiones y cuál es la última?</strong>
            <p>HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Fue creado por Tim Berners-Lee en 1991. Las versiones principales han sido HTML 2.0, HTML 3.2, HTML 4.01, XHTML y HTML5. La última versión es HTML5, publicada en octubre de 2014.</p>
        </li>
        <li>
            <strong>¿Cuáles son los principios básicos que el W3C recomienda seguir para la creación de documentos con HTML?</strong>
            <p>El W3C recomienda seguir principios como la semántica (usar etiquetas adecuadas para el contenido), la accesibilidad (hacer contenido accesible para todos los usuarios), la separación de contenido y presentación (usar CSS para estilos) y la compatibilidad con estándares.</p>
        </li>
        <li>
            <strong>En las Especificaciones de HTML, ¿cuándo un elemento o atributo se considera desaprobado? ¿y obsoleto?</strong>
            <p>Un elemento o atributo se considera desaprobado cuando su uso ya no es recomendado y puede ser eliminado en futuras versiones. Se considera obsoleto cuando ya no es soportado por los navegadores y ha sido eliminado de las especificaciones.</p>
        </li>
        <li>
            <strong>¿Qué es el DTD y cuáles son los posibles DTDs contemplados en la especificación de HTML 4.01?</strong>
            <p>El DTD (Document Type Definition) define la estructura y las reglas de un documento HTML. En HTML 4.01, los posibles DTDs son: Strict, Transitional y Frameset.</p>
        </li>
        <li>
            <strong>¿Qué son los metadatos y cómo se especifican en HTML?</strong>
            <p>Los metadatos son información sobre el documento, como el autor, la descripción o las palabras clave. Se especifican en la sección <code>&lt;head&gt;</code> usando etiquetas como <code>&lt;meta&gt;</code>.</p>
        </li>
    </ol>
